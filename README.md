# Vibe Coding Projects

Three interactive web experiments built with **Claude Code** — no programming experience required.
Each file is self-contained HTML. Just open it in a browser.

---

## 01 · Interactive Eye-Following Card

`01-eye-following-card.html`

A card with two eyes that follow your mouse cursor in real time.
Cloned from a [Figma community file](https://www.figma.com/community/file/1530305855415503769) using only natural language instructions.

**Techniques:** CSS · SVG · Mouse Events · Figma → Code

---

## 02 · Image Reveal Gallery

`02-image-reveal-gallery.html`

Hover over the image to reveal it through a circular lens.
Double-click to expand the full image with a smooth animation.

**Techniques:** CSS clip-path · Blur & Blend modes · Double-click interaction

> Inspired by [this component on 21st.dev](https://21st.dev/community/components/jatin-yadav05/image-reveal/default)

---

## 03 · Holographic Pokémon Card

`03-holographic-pokemon-card.html`

A 3D card that tilts with your mouse, with a rainbow foil shimmer and diamond checker metallic texture — like a real holographic trading card.

**Techniques:** CSS Blend Modes · 3D Transform · Repeating Gradients · Dynamic Shadows

> Claude's own suggestion when asked: *"what should we build next?"*

## How it was made

Everything was built in a single Claude Code session through conversation — no code was written manually.
The workflow: describe → preview → refine → repeat.

**This is Agentic Coding.** If something makes you say "wow", you already have everything you need to build it.
